# Ligeniqo Website

Static marketing website for Ligeniqo, built as a custom code rewrite of the original WordPress site.

## What is included

- Home page
- About page
- Services overview
- Lead generation proposal page
- Supporting service subpages

## Stack

- HTML
- CSS
- JavaScript

## Local use

Open `index.html` in a browser or serve the project with any static file server.

## Notes

- Brand colors and typography follow the Ligeniqo visual system.
- Temporary working files are ignored via `.gitignore`.
