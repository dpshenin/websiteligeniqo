# Ligeniqo Skills

This file is the quick reference for the Ligeniqo site.

Canonical guide:

- [SKILLS.md](/Users/dpshenin/Website%20Ligeniqo/SKILLS.md)

Short version:

- Keep the site in the Ligeniqo brand system.
- Main font: `Lato`.
- Main colors: `#0b0aa3`, `#02d2fe`, `#111111`, `#425466`, `#616161`, `#eef0ff`, `#f1f7fb`, `#fbf8f1`.
- Prefer white space, big headings, clean cards, and editorial layouts.
- Use original designer assets when they exist.
- Do not add repeated Ligeniqo logos inside content sections unless the source design does that.
- Lead Generation should read like a strategic proposal: context, system, economics, and rollout.
- Keep interactivity useful but restrained: tabs, carousels, marquee, and modal CTA are fine.
